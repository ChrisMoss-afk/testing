# Polytechnic Pulse V2.25 Content Editor Guide

This prototype is designed so a non-coder can update the hub by editing plain `.txt` files inside the `content/` folder.

## Rule
Each content item lives between `---` dividers. Keep the field names the same. Replace the text after the colon.

## Main Files

- `alerts.txt` controls the alert rail.
- `news.txt` controls campus updates.
- `events.txt` controls the event timeline.
- `deadlines.txt` controls date-sensitive reminders.
- `resources.txt` controls quick links.
- `videos.txt` controls Watch Campus cards.
- `jobs.txt` controls the Work Board.
- `sponsors.txt` controls sponsor cards.
- `ad-packages.txt` controls sellable sponsor and employer products.
- `partners.txt` controls partner/logo slots.

## Job Board Fields

Use `job_type` for the filter category: `part-time`, `seasonal`, `internship`, `volunteer`, or `entry-level`.

Use `field` for the program/industry filter: `business`, `trades`, `health`, `technology`, `hospitality`, or `community`.

## Governance

Before a public launch, every listing should be reviewed for accuracy, approval, audience fit, and whether it is editorial, partner-submitted, student-submitted, or paid.
